<!--more-->
根据自带的教程，进入到hexo安装目录下
```
npm install --save hexo-admin
hexo server -d
```
打开http://localhost:4000/admin/    

不需要专门的编辑器且可以实现写出hexo效果完全相等的文档。    
以前用的几个编辑器会出现MD标签效果不一样的情况。
